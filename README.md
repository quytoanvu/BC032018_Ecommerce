# BC032018_Ecommerce
Requirement:
- Truffle webpack installation - https://github.com/trufflesuite/truffle-init-webpack
- IPFS - https://ipfs.io/docs/install/

Launching:
- IPFS
```
    ipfs daemon
```
- Truffle
```
    truffle develop
```
- Launch website
```
    npm run dev
```

Testing:
```
    truffle develop
    truffle test
```
